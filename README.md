# React + NestJS Blog App

Minimalistic but fully functional blog application

## Technologies

- React (Frontend)
- Next (BFF)
- NestJS (Backend)
- Prisma (ORM)
- PostgreSQL (DB)
- Nginx (Exposing both frontend and backend services in single domain space (and also static files serving))
- Docker
